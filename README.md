# BitcoinApplication

minSdkVersion 25
<br>
targetSdkVersion 34

USED
- MVVM architectural
- Data Binding , Compose , 
- dependency injection : koin
- Local Database : Realm
- RecyclerView
- Multi Module : App , Fibonanci , Prime Number , Recyclerview , Validate Pincode , Number Picker
- Unit Test
- Call API using : Retrofit
